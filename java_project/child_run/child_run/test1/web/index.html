我们是谁？

"数智童科启蒙团"是一个专注于乡村儿童计算机科学素养提升的公益项目。我们发现，许多欠发达地区的校舍虽然配置了电脑，但孩子们缺乏系统的指引去理解科技。

我们通过制作通俗易懂的动画、组织线上互动教学，让复杂的二进制、算法、硬件知识变成孩子们眼中的"魔法"。

  • ✅ 普及化： 零基础起步，适配儿童认知
  • ✅ 趣味化： 拒绝说教，寓教于乐
  • ✅ 持续性： 建立长期更新的资源库
